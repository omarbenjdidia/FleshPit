Installation

To add Flexalon to your project:

1. Import the "Flexalon" directory and optionally the "Flexalon Samples" directory into the "Assets" folder in your Unity project.

2. Create an empty GameObject and add the "Flexalon" singleton component to it. Alternatively, this singleton will be created automatically when you try to use any Flexalon Component.

Well, that was easy! Head over to Core Concepts to learn what Flexalon can do!

Download the new Flexalon Template Pack for complete examples you can copy into your scene.
